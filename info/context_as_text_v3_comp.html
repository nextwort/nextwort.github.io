context_as_text_v3_comp

This is basicaly the same as context_as_text_v3 but it is used to complete the current word.

For this the model runs context_as_text_v3 to get a list of all models that can come after the current word.

Then the code iterates over this list and returns the first 3 occurances that start with the beginning of the word that has to be completed.
